# 诗词可视化 RAG 系统 — 项目进度细节技术说明

> 记录时间：2026-03-04  
> 当前状态：**本地三服务联调可用**，AI Pipeline 关键模块为占位实现，尚未接入真实向量检索与扩散模型

---

## 一、整体架构

```
浏览器 (Vue 3 / Vite :5173)
    ↕ Vite dev proxy /api → :8080
    ↕ Vite dev proxy /ai  → :8000
后端 (Spring Boot :8080)
    ↕ HTTP REST 回调
AI 服务 (FastAPI / uvicorn :8000)
    ↕ (占位) 内存检索 + GLM-5 (可选) + 占位 Diffusion
MySQL 8.0 (:3306, 库名 poetry_rag)
```

|------|------|------|
| 向量检索 | `app/modules/retriever.py` | **占位**：当前硬编码返回"山水、孤帆、夕阳"，需接入 ChromaDB / FAISS + 真实诗词语料 |
| 扩散模型推理 | `app/modules/generation.py` | **占位**：返回随机 UUID 假路径 `/statics/outputs/{uuid}.png`，需对接 Stable Diffusion WebUI API |

### Pipeline 执行流程（已实现的逻辑框架）

```
接收诗句 sourceText
   → Retriever.fetch()      (占位: 硬编码意象)
   → PromptEnhancer.enrich() (GLM-5 增强 or fallback 拼接)
   → DiffusionClient.generate() (占位: 返回假图片路径)
   → HTTP 回调后端 /api/v1/poetry/callback
```

## 三、后端（Spring Boot 3 / Java 17）

### 已完成

#### 3.1 用户认证模块

| 类 | 说明 |
|----|------|
| `AuthController` | `POST /api/v1/auth/register`、`POST /api/v1/auth/login` |
| `AuthService` | 注册（BCrypt 加密密码 + 用户名去重）、登录（密码校验）|
| `TokenService` | 基于内存 `ConcurrentHashMap` 的 Token 会话，UUID Token，可配置过期时间（默认 720 分钟）|
| `AuthTokenInterceptor` | 拦截 `/api/v1/**` 所有请求，放行 `/auth/**`、`/callback`、`/actuator`，其余需 `Bearer` Token |

#### 3.2 诗词任务模块

| 类 | 说明 |
|----|------|
| `PoetryVisualizationController` | `POST /api/v1/poetry/visualize`（提交）、`GET /api/v1/poetry/task/{taskId}`（轮询）、`POST /api/v1/poetry/callback`（AI 回调） |
| `TaskDispatchService` | 创建任务写库 → 异步 HTTP 派发到 AI 服务 → 失败时更新状态为 FAILED；接收回调 → 更新检索文本/提示词/图片 URL |

#### 3.3 基础设施 / 配置

| 类 | 说明 |
|----|------|
| `CorsConfig` | CORS 允许来源读自 `app.security.cors.allowed-origins`，开发默认 `http://localhost:5173` |
| `AiServiceProperties` | 绑定 `ai.service.url` / `callback-url` / `callback-token` 三项配置 |
| `AppSecurityProperties` | 绑定 Token TTL、CORS 白名单 |
| `MyBatisPlusMetaHandler` | 自动填充 `gmtCreate` / `gmtModified` |
| `GlobalExceptionHandler` | 统一捕获 `IllegalArgumentException`（400）及兜底 500 |

#### 3.4 数据库实体

| 表 | 实体类 | 说明 |
|----|--------|------|
| `sys_user` | `User` | 用户表（id, username, password BCrypt, nickname, avatar, status）|
| `sys_generation_task` | `GenerationTask` | 任务表（task_id UUID, original_poem, retrieved_text, enhanced_prompt, result_image_url, task_status 0/1/2）|
| `sys_favorite` | `Favorite` | 收藏表（建表 SQL 已有，**Mapper/Service/Controller 尚未实现**）|

#### 3.5 任务状态机

```
PENDING (0)  →  COMPLETED (1)
             →  FAILED    (2)
```

### 尚未实现

| 功能 | 说明 |
|------|------|
| 收藏功能 API | `sys_favorite` 表已建，后端 Mapper/Service/Controller 一层都未写 |
| 历史记录持久化 API | 前端历史页面数据来源为 Pinia 内存 Store，刷新即丢失；后端无分页查询历史接口 |
| 任务重试机制 | AI 派发失败置为 FAILED 后无自动重试逻辑 |
| Token 持久化 | TokenService 使用内存 Map，重启服务后所有已登录 Token 失效 |
